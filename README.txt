== Installing In Theatro Veritas Notepad Manually (FTP) ==

	To add an In Theatro Veritas Notepad to your WordPress installation, follow these basic steps:
		1. Download the In Theatro Veritas Notepad archive and extract the files it contains. You need to preserve the directory structure in the archive when extracting these files. Using an FTP client to access your host web server, create a directory to save In Theatro Veritas Notepad in the wp-content/themes directory provided by WordPress. For example: wp-content/themes/theatro-notepad
		2. Upload the In Theatro Veritas Notepad files to the new directory on your host server.
		3. Follow the instructions below for selecting the In Theatro Veritas Notepad.

	To select In Theatro Veritas Notepad for your website:
		1. Log in to the WordPress Administration Panel.
		2. Select the Appearance panel, then Themes.
		3. From the Available Themes section, click on the Theme title (or Theme screenshot) for In Theatro Veritas Notepad to activate.
		4. A preview of the Theme will be shown.
		5. To activate the Theme click the Activate "In Theatro Veritas Notepad" link in the top right.
		6. Then you need to go to the Appearance —> Customize for further customisation of your theme.

	Your selection should immediately become active.

	Note: If the Theme preview is blank, do not activate the new Theme without investigating further. Your site may not be displayed correctly, otherwise.

== Customizing In Theatro Veritas Notepad ==

	To customise In Theatro Veritas Notepad using the built in options, In Theatro Veritas Notepad should already be active:
		1. Log in to the WordPress Administration Panel.
		2. Select the Appearance panel, then Customize.
		3. A page explaining the options will be displayed.
		4. Click save to finalise your selections.

	Your selection should immediately become active.
